# 🛒 Retail Sales Analysis

## 📌 Project Overview

This project is an End-to-End Retail Sales Analysis built using MySQL, Python, Pandas, Matplotlib, and Power BI.

The objective is to analyze customer behavior, sales performance, payment methods, product categories, and delivery performance to generate business insights.

---

## 🛠 Tech Stack

- MySQL
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Power BI
- DAX

---

## 📂 Dataset

The project contains the following datasets:

- Customers
- Orders
- Order Items
- Products
- Payments
- Reviews

---

## 📊 Project Workflow

CSV Files

↓

MySQL Database

↓

SQL Business Queries

↓

Python Data Analysis (Pandas)

↓

Power BI Dashboard

---

## 📈 Business Questions Solved

- Monthly Revenue Trend
- Revenue by State
- Revenue by Payment Type
- Top 10 Product Categories
- Top Customers
- Average Delivery Days
- Review Score Analysis
- Monthly Orders Trend

---

## 📌 Key Insights

- Credit Card is the most used payment method.
- São Paulo generates the highest revenue.
- Beauty & Health is the highest revenue category.
- Average Delivery Time is approximately 12 days.
- Average Review Score is around 4.03.
